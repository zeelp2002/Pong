## Instructions
Player 1: 'Up' key to go up
          'Down' key to go down
          
Player 2: 'A' key to go up
          'Z' key to go down
          
Press 'r' key to reset game 
          
## Rules
The objective it not let the ball leave the screen on the left and right sides
Player that reaches 7 points first is the winner
